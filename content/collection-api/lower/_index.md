Return a new string in which all case-based characters are in lower case.

This function does *not* generate an [event](../../events).

### Function
*string*.`lower()`

> Example using ***lower()***:

```
'Hello World!!'.lower();
```

> Return value in JSON format

```json
"hello world!!"
```
