Operator | Description
-------- | -----------
`+` | Addition operator.
`-` | Subtraction operator.
`/` | Float division operator.
`//` | Integer division operator.
`*` | Multiplication operator.
`%` | Modulo operator.

> Arithmetic examples:

```
[
    5 + 2,
    5 - 2,
    5 / 2,
    5 // 2,
    5 * 2,
    5 % 2,
];
```

> Return value in JSON format

```json
[
    7,
    3,
    2.5,
    2,
    10,
    1
]
```
