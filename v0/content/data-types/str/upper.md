Return a new [string](..) in which all case-based characters are in upper case.

This function does *not* generate an [event](../../../events).

### Function

*str*.`upper()`

> Example using ***upper()***:

```thingsdb,json_response
'Hello World!!'.upper();
```

> Return value in JSON format

```json
"HELLO WORLD!!"
```
